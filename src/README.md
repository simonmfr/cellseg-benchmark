# User guide

## cell_annotation.py
requires sopa installation. Arguments:
1) sample name
2) method name
3) data directory
4) mad_factor: if number smaller than 1, then defaults to 3

Usage:
```
python cell_annotation.py sample_name method_name data_dir mad_factor
```

## transcript_tif.py
Arguments:
1) data path

Usage:
```
python transcript_tif.py data_path
```
